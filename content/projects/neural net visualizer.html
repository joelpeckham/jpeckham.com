About

This neural network is a simple classifier. It takes 5x5 pixel images as input, and outputs the digit that each image represents. All 25 pixels of the image are fed into the neural network as input nodes. The output is a 10 node layer, where each node represents a digit from 0 to 9. Nodes with stronger activations appear lighter in color, therefore the output node with the brightest color is the digit that the neural network thinks the image represents. As the neural network trains, the accuracy of the network increases.
